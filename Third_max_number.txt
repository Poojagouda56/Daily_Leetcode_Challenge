Given an integer array nums, return the third distinct maximum number in this array. 
If the third maximum does not exist, return the maximum number.

solution in python3 =>

class Solution:
    def thirdMax(self, nums: List[int]) -> int:
        a=set(nums)
        if len(a)<3:
            return max(a)
        else:
            a=list(a)
            a.sort()
        return a[-3]
